>> To run the Enhancing Digital Certificate Security through Blockchain and Smart Contracts Project, you need to have a node installed on your computer. We recommend Node for the same. Download Node Using Link:
NODE:
https://nodejs.org/dist/v20.11.0/node-v20.11.0-x64.msi

Once you have Node running, follow these steps:
1. Extract the downloaded file.
2. In Folder Use command 'npm start' / 'start npm' to install node modules.
3. Open the application in your browser using Liver Server Extension in VS Code.
4. The owner of the system must first add an exporter to the list of authorized parties. This is done by clicking on the "Add Exporter" button and entering the exporter's Ethereum address.
5. Upload a document to the system by clicking on the "Upload Document" button and selecting a file from your computer. The document will be encrypted and stored in the IPFS network, and its hash will be recorded in the Blockchain.
6. Verify a document by clicking on the "Verify Document" button and entering its unique identifier (hash) in the input field. The system will retrieve the document from the IPFS network, decrypt it, and compare its hash with the one recorded in the Blockchain.
7. The system will display a message indicating whether the document is authentic or not.
